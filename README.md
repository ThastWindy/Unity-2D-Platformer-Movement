# Unity-2D-Platformer-Movement
### Unity 2D Platformer Movement
Including: 
- Movement
- Jumping
- Wall Jump

Component Requires:
- RigidBody2D
- Any Collider

Player Requires:
- GroundCheck
- WallCheck

Step:
1. Create Player
2. Create GroundCheck + WallCheck
3. Create and Select Ground and Wall Layer
